## 差动保护页面崩溃修复
- 问题：点击"填入默认值"或填写电压后页面消失
- 根因：`fillTransformerDefaults` 对响应式 `reactive` 对象使用整体赋值（`params.voltages = { hv, lv }`），破坏 Vue 响应式链接
- 修复：改为逐个属性赋值（`params.voltages.hv = 110` 等）
- 同时将 `transformerStore` 改为纯 `computed` 模式，移除 `recalculate` action
