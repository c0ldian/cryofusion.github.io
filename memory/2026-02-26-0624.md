## 模块设计

### 路由
- `/calculators/distance` → 距离保护导航页
- `/calculators/distance/phase` → 相间距离校验
- `/calculators/distance/directional` → 方向距离校验

### 整定参数（市面常用）
**相间距离校验**
- Z1 (Ω) - 正序阻抗定值
- t1 (s) - 动作时间
- U_secondary (V) - 电压二次额定值，默认 100
- I_secondary (A) - 电流二次额定值，默认 5
- CT 变比、PT 变比（可选，用于一次值换算）

**方向距离校验**
- Z_set (Ω) - 阻抗整定值
- θ_sens (°) - 方向灵敏角，常用 -30° (滞后) 或 +30° (超前)
- K_offset - 偏移系数（方向特性），常用 0.1~0.2
- t_set (s)
- U_secondary, I_secondary
- CT/PT 变比

### 测量配置
- 故障类型：AB、BC、CA（两相短路）
- 测量点位置：首端 / 中段 / 末端（默认中段）
- 是否模拟线路阻抗：是/否
  - 线路单位长度阻抗 Z'/km (Ω/km)
  - 线路总长 L (km)
  - 测量点距首端距离 x (km)，默认 L/2
- 负载条件：前送负荷功率（可选）

### 计算引擎
**相间距离**
- 测量阻抗：Z_meas = (U_A - U_B) / I_A（或 I_B）
- 动作判据：|Z_meas| ≤ Z1 且 t ≥ t1
- 结果：动作/不动作，测量阻抗值，动作时间

**方向距离**
- 引入方向判据：方向元件的动作特性（偏移圆、特性角）
- 常用方向特性：方向阻抗圆，偏移方向 θ_sens
- 根据灵敏角计算正向/反向边界：调整 U/I 相位，找到刚好动作/不动作的边界
- 输出边界角度范围，验证方向正确性

### 输出内容
调试方案包含：
1. **状态序列表**：每次试验的 U相、V相、W相电压（幅值、相角），A、B、C相电流（幅值、相角）
2. **物理接线**：调试台通道 → 装置端子映射，CT/PT 极性
3. **关键公式**：测量阻抗计算公式，方向判据公式
4. **安全注意事项**

### 默认工程值
- 线路：110kV，Z0≈0.4Ω/km，L≈30km
- 测量点：L/2 → x=15km
- 故障电流：3kA（一次值）
- 电压二次：100V，电流二次：5A
